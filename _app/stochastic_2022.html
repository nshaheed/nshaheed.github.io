Hello,

Here is a portfolio of some of my audiovisual works I thought would be relevant to my application. The first two works (In this place. and op breath) utilized Pix2PixHD to create the visual material. The final piece, Wire Spill, uses more traditional generative animation techniques.

Thanks,

Nick Shaheed

In this place.

In this place is an audiovisual work. The visual material is created using pix2pixHD. Drone footage of a eucalyptus tree was used to create a frame prediction model. Because the model only has a memory of 1, it is a poor frame predictor, but creates interesting shapes and textures.

op breath

op breath is an audiovisual work. The visual material is created using pix2pixHD, where each of the six squares in the grid is the same source data inferred using a different epoch from the training stage.

Wire Spill

Score

In addition to the electronic audio, this work uses a live "video synthesizer" created in Processing. Various parameters are tweaked to modify the animation as part of the performance to visually navigate through the high-dimensional space defined in the program.
