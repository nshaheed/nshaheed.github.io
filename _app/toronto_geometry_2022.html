op breath - Electronics and Visuals

op breath is an audiovisual work. The visual material is created using pix2pixHD, where each of the six squares in the grid is the same source data inferred using a different epoch from the training stage.

Wire Spill - Kalimba, Electronics, and Generative Animation

Score

In addition to the electronic audio, this work uses a live "video synthesizer" created in Processing. Various parameters are tweaked to modify the animation as part of the performance to visually navigate through the high-dimensional space defined in the program.

Chaotic Substrate - Live Electronics and Video

Score

Chaotic Substrate is a structured audiovisual improvisation. Both the pitch and visual content are derviced from the logistic map as the recurrent function is applied to infinity.
